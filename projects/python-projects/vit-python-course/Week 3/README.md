# Rick and Morty Data Explorer

## Overview

This project was completed as part of the Python Fundamentals Bootcamp offered by [Vets In Tech](https://vetsintech.co/) and sponsored by [DraftKings](https://www.draftkings.com/).

The code retrieves data from the Rick and Morty API and exports the results into a multi-sheet Excel workbook using Python and OpenPyXL. The exercise provided hands-on experience working with APIs, JSON data, Python data structures, and spreadsheet automation.

This work represents an early milestone in my Python learning journey and helped build the foundation for more advanced data-processing and reporting projects completed later in this course (Please see: State Travel Report python project in the Week 4 folder).

## Features

- Retrieves data from multiple Rick and Morty API endpoints
- Processes JSON responses into Python objects
- Creates an Excel workbook automatically
- Generates separate worksheets for:
  - Characters
  - Locations
  - Episodes
- Dynamically creates worksheet headers based on API data
- Exports structured data into a format suitable for analysis and reporting
