# SHA-256 Hashing Demo (Python Cybersecurity Self-Directed Lab)

## Overview

This is a self-directed, hands-on project demonstrating SHA-256 cryptographic hashing using Python’s built-in hashlib library.

This project was built as part of my learning journey in both Python programming fundamentals and CompTIA Security+ (SY0-701) certification studies, specifically aligned with:

    CompTIA Security+ (SY0-701) Exam Objectives
    Domain 1: General Security Concepts
    Objective 1.4: Cryptographic Solutions

While studying the Security+ exam objectives, I wanted to reinforce the concept of hashing through practical application. This project was created as a way to translate theoretical cybersecurity concepts into a working Python project.

Additionally, this project builds upon Python programming fundamentals learned in the Vets In Tech Python Fundamentals Bootcamp (May 2026 cohort). Since completing that bootcamp, I have continued practicing independently by building additional projects to strengthen my understanding.

## Features

- SHA-256 cryptographic hashing using Python’s built-in hashlib module
- Conversion of plaintext strings into byte format for cryptographic processing
- Generation of fixed-length hash output (digest)
- Display of both original input and resulting hash value
- Demonstrates a foundational cybersecurity concept: data integrity through hashing

## Technologies Used

- Python 3
- hashlib (Python Standard Library)

## Skills Demonstrated

- Python programming fundamentals (variables, functions, string handling)
- Using built-in Python security libraries (hashlib)
- Understanding cryptographic hashing concepts (SHA-256)
- Converting strings to byte format for processing
- Generating and interpreting hash digests
- Structuring a simple Python script for cybersecurity use cases
