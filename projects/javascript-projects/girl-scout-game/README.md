# Girl Scout Cookie Adventure Game

## Overview

The Girl Scout Cookie Adventure Game is a JavaScript-based interactive browser game designed as an exploratory project in game logic, state management, and DOM-based user interaction.

This project began as a text-based JavaScript game and evolved into an early-stage web-based UI application as part of my self-directed learning in frontend development.

The current version demonstrates core programming logic and user interaction flow, and reflects an ongoing transition from console-style scripting to browser-based application development.

## Project Goal

The objective of the game is to simulate a simple sales-driven decision loop where the player:

- Manages cookie inventory
- Maintains energy levels
- Responds to customer interactions
- Attempts to maximize sales before running out of resources

## Technologies Used

- JavaScript (ES6 fundamentals)
- HTML5
- CSS (external shared stylesheet)
- DOM manipulation
- Event-driven programming

## Core Concepts Practiced

- Game loop and state management
- Conditional logic and branching outcomes
- Randomized event generation
- User input handling via forms and buttons
- DOM updates for dynamic UI feedback
- Basic UI structure and event listeners

## Project Evolution

This project began as a fully text-based JavaScript game and was later refactored into a browser-based interface using HTML and DOM manipulation.

During development, the focus shifted toward:

- Translating text-based prompts into UI elements
- Managing game state across user interactions
- Experimenting with event-driven architecture in the browser

This transition is still in progress, and the project remains an evolving learning exercise.

## Current Status
