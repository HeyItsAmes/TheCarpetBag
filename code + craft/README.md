## Introduction

Code + Craft is a professional services website for web design, web development, graphic design, and technology solutions.

The site provides information about creative and technical services, project experience, and technology-focused solutions for businesses, organizations, and individuals.

## Services

Services may include:

- Website design and maintenance
- Small business and nonprofit websites
- Digital graphics and web assets
- Logo and brand identity design
- Technical documentation and training materials
- Content management and website administration
- AI-enhanced workflows and productivity solutions
- Project coordination and technology consulting
